Usuários:

Listar Todos os Usuários: (GET)

https://cookmastercrud.herokuapp.com/getall

Cadastrar um usuário: (POST)

https://cookmastercrud.herokuapp.com/users/

Body: { "name" : "luiz", "email": "luiz@gmail.com", "password": "123" }

Efetuar Login: (POST)

https://cookmastercrud.herokuapp.com/login

Body: { "email": "luiz@email.com", "password": "admin" }

Um token será gerado e ficará ativo por 15min.

Deletar seu próprio usuário: (DELETE)

https://cookmastercrud.herokuapp.com/deleteuser/id_do_usuario

Regra: Admins podem deletar qualquer conta. Usuários só podem deletar a própria conta.
