Bem vindo(a) a API Cookmasters!

Como utilizar?

GET: As rotas com método GET podem ser acessadas diretamente no navegador.

POST, DELETE ou PUT: Deverá ser utilizado o Insomnia ou o Postman, passando no Body da requisição um arquivo .JSON com o objeto descrito na rota.

*Algumas rotas necessitam do Token de autenticação: inserir no campo 'authorization' do cabeçalho.


Endpoints Disponíveis:

Usuários:

Listar Todos os Usuários: (GET)

https://cookmastercrud.herokuapp.com/getall

- Cadastrar um usuário: (POST)

https://cookmastercrud.herokuapp.com/users/

Body: { "name" : "luiz", "email": "luiz@gmail.com", "password": "123" }

Efetuar Login: (POST)

https://cookmastercrud.herokuapp.com/login

Body: { "email": "root@email.com", "password": "admin" }

Deletar Todos os Usuários: (DELETE)

https://cookmastercrud.herokuapp.com/deleteall

Receitas:

Listar todas as Receitas: (GET)

https://cookmastercrud.herokuapp.com/recipes

Listar uma Receita por ID: (GET)

https://cookmastercrud.herokuapp.com/recipes/id_da_receita

Cadastrar uma Receita: (POST)

https://cookmastercrud.herokuapp.com/recipes

Body: { "name": "Mingau de Aveia",
"ingredients": "Aveia, Banana, Leite e Canela",
"preparation": "Ferva o leite e acrescente todos os ingredientes."}

Atualizar uma Receita por ID: (PUT)

https://cookmastercrud.herokuapp.com/recipes/id_da_receita_a_ser_alterada

Body: { "name": "Mingau de Aveia com Canela",
"ingredients": "Aveia, Banana, Leite e Canela",
"preparation": "Ferva o leite e misture todos os ingredientes por 10min."}

Deletar uma Receita por ID: (DELETE)

https://cookmastercrud.herokuapp.com/recipes/id_da_receita_a_ser_deletada

Adicionar uma Imagem à Receita: (PUT)

https://cookmastercrud.herokuapp.com/recipes/id_da_receita/image

(OBS: Upload deve ser do tipo Multipart form Data)
